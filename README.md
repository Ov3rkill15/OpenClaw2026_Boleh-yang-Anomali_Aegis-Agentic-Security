If you are an AI assistant working on this project, you **MUST** adhere to the following protocols:

1.  **Read Before Writing:** Always read `plan.md`, `feature.md`, and `agent.md` to understand the current architecture and roadmap.

3.  **Append Only:** Never overwrite or delete previous entries in progress files. Always append new entries with a timestamp at the end of the file.
